# [Demo Restful API](https://www.springboottutorial.com/spring-boot-react-full-stack-crud-maven-application)
## Frontend
- ### ReactJs
## Backend
- ### Springboot

## Use
```powershell
git clone https://github.com/nguyentrinhan-dev/demo-reactwithrestfulAPI.git
cd frontend
npm install
```
Start backend project, goto `http://localhost:8080/ins/nguyentrinhan/courses` to confirm

Start front end
```powershell
npm start
```
